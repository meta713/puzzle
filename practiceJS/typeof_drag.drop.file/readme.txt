・ファイルの選択はimageとvideoが選択できてimageを選択すると背景変わります。背景はストレージに格納されてページをリロードしても
背景を保持します。ストレージには基本的に4,5MBくらい保存できたと思うんやけど、あれやったらuk.jsの先頭らへんにあるlocalStorage.clear();
をコメントインしてストレージを綺麗にしてほしい。(clearするとストレージ内のデータが全て削除されるから嫌なら localStorage.removeItem(background)
を実行してほしい、backgroundデータだけ消えるから)

・テキストは基本的にドラッグ・ドロップできる。Drop hereにドロップするとDrop hereのテキストがドロップされたテキストに変換されます。
モーダルのテキストも変換されるんで。

・network is on、network is offは現在の環境を表していて、ネットに繋がってるかどうかを示しています。
